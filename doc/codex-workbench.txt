==============================================================================
OVERVIEW                                           *codex-workbench-overview*

codex-workbench.nvim runs Codex in a shadow git worktree. Codex edits are kept
out of the real worktree until the user accepts review patches from Neovim.

The runtime has no required Lua plugin dependencies. A Rust bridge handles
Codex app-server JSONL, shadow worktree sync, patch generation, and apply.

==============================================================================
COMMANDS                                           *codex-workbench-commands*

:CodexWorkbenchOpen
  Start the bridge and open the output pane.

:CodexWorkbenchAsk [prompt]
  Choose a Codex thread for the current repository, then send a prompt.
  Without [prompt], opens vim.ui.input() after thread selection.

:CodexWorkbenchReview
  Open the pending review queue.

:CodexWorkbenchThreads
  Show Codex threads for the current repository and resume the selected thread.

:CodexWorkbenchAccept [scope]
  Apply the pending review to the real workspace. Scope defaults to "all".
  Supported scope forms: all, file:path, hunk:path:index.

:CodexWorkbenchReject [scope]
  Reject the pending review scope. Scope defaults to "all".

:CodexWorkbenchAbandon
  Mark the pending review rejected and resync shadow from the real workspace.

:CodexWorkbenchResume [thread_id]
  Resume the stored thread or the provided thread id.

:CodexWorkbenchFork
  Fork the current Codex thread.

:CodexWorkbenchStatus
  Print bridge status.

:CodexWorkbenchToggleDetails
  Toggle raw app-server event rendering in the output pane.

:CodexWorkbenchLogs
  Open the codex-workbench log file.

:CodexWorkbenchHealth
  Run health checks.

:CodexWorkbenchInstallBinary
  Print the install script path for the Rust bridge binary.

OPTIONS                                            *codex-workbench-options*

>
require("codex_workbench").setup({
  codex_cmd = "codex",
  binary = { auto_install = false },
  ui = {
    output = { position = "right", size = 40 },
    review = { layout = "vertical" },
  },
  session = { auto_resume = true },
  shadow = {
    root = vim.fn.stdpath("state") .. "/codex-workbench/shadows",
    max_untracked_file_bytes = 5242880,
    max_untracked_total_bytes = 52428800,
  },
  keymaps = {},
  statusline = { enabled = true },
})
